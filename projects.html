Wordle Project A terminal-based implementation of the popular Wordle game using Python. This project challenges players to guess a five-letter word within five attempts, providing feedback for each letter (correct, misplaced, or incorrect). It features input validation, word dictionary integration, and a clean command-line interface, making it a fun and educational exercise in string handling, logic, and user interaction. Python
Suburb Explorer APP A cross-platform mobile application developed using .NET MAUI that allows users to search and view details about suburbs in Australia. The app includes features such as suburb statistics and suburb scores. Designed with a responsive UI and MVC architecture and integrates real-time data from ABS API to enhance the user experience. C#, .NET MAUI, REST API
Portfolio Website A personal portfolio website built with HTML, Tailwind CSS, and JavaScript to showcase projects, skills, and experience. Designed with a mobile-first layout and smooth UI interactions. HTML, Tailwind CSS, JavaScript, Ninja API Link to my Portfolio page
Moondalup Car Park A smart car park management system developed to monitor parking availability in real time. Built using Python and MQTT protocol, the system simulates sensor-based tracking of vehicles entering and exiting the parking area. It features live data updates and occupancy status for IoT environments. This project demonstrates the integration of messaging protocols with backend logic for efficient, event-driven communication. Python, MQTT
Open Source Project A OCR tool to extract and recognize code from video tutorials, with a focus on accessibility for visually impaired developers. OcrRoo features a custom video player that analyzes video frames, detects code snippets using Optical Character Recognition (OCR), and reads the code aloud or provides structured output. This tool bridges the gap between visual learning resources and accessible education, making coding tutorials more inclusive and navigable for all users. Python
